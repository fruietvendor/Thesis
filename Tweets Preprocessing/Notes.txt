The code files in this folder are for
1) Preprocessing the tweets
2) Training the word2vec model used for sentiment analysis
3) Running sentiment analysis on the cleaned tweets 
4) Identifying muslim related tweets, and creating the final dataset for the changepoint analysis part
5) Creating the weekly corpora to train the weekly word2vec models which will help in the contextual analysis

Make sure that this folder contains:
1) classifier.pkl 
2) emoji vectorizer.pkl 
3) muslim_terms.pkl
4) slang.txt
5) wv_model.pkl

Download the folder "Raw Data" from the Drive link and extract it in the "Raw Data" folder in this main folder
Download the folder "Cleaned Data" from the Drive link and extract it in the "Cleaned Data" folder in this main folder, or run the "preprocessing tweets" notebook to populate it.
